# StayFlex

StayFlex is a responsive hotel booking application developed by Rohit Arabale**.

## Project Overview

StayFlex helps users discover hotels, view destination options, check hotel details, and book flexible hourly stays. The project focuses on a smooth, mobile-friendly booking experience built with modern frontend tools.

## Developer

- Developer: Rohit Arabale
- **Project:** StayFlex hotel booking app
- **Role:** Frontend development, UI implementation, responsive experience, and booking flow structure

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- React Datepicker
- React Icons
- Framer Motion
- Auth0 React SDK

## Features

- Fully responsive design for mobile, tablet, and desktop screens
- Hourly hotel booking concept for flexible stays
- Destination browsing and hotel listing pages
- Hotel detail view with amenities, reviews, and booking actions
- FAQ and support pages
- Authentication-ready login and signup flow

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
